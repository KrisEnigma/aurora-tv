Aurora is a fork of Moonlight TV for LG webOS, tuned for high-bitrate 4K/120Hz HDR streaming from a Sunshine or NVIDIA GameStream host, with an AMOLED-friendly dark theme and a remote- and gamepad-first UI.

Highlights

  • AMOLED layout — pure black background, dark surfaces, violet accent across the launcher, game grid, and settings.
  • 3.6K resolution (3584×2016) — ~90% of 4K pixel area with less decoder load and lower input lag than native 4K on recent LG panels.
  • HDR10 (PQ) over HEVC Main10 or AV1 Main10 when the host supports it.
  • NTSC-correct refresh rates (59.94/119.88 Hz for 60/120 FPS) for smoother sync on LG panels.
  • Periodic decoder refresh for HEVC, avoiding long-session artifact drift.

Controllers & HID Passthrough

Bridges DualSense, Xbox, Steam Controller, Flydigi/Vader3, and generic pads to the PC as native USB devices via CTM-USBIP, so the host sees the real controller instead of a generic virtual pad.

  • Per-controller Auto-Connect toggle — choose passthrough or a regular virtual controller per pad, handy for streaming with two controllers at once.
  • Auto-plug on stream start plus hotplug support.
  • DualSense-specific tuning: feedback-latency pacing, input coalescing, battery level in the HID overlay.
  • HID panel is fully gamepad-navigable.

Built for TV remotes and gamepads

  • On-screen streaming overlay with live latency/bitrate stats, opened with the remote's red button or a gamepad combo.
  • Full soft keyboard (Windows/Xbox OSK style) with sticky modifiers, openable directly from the gamepad.
  • Virtual mouse mode for desktop apps and launchers — right stick moves the cursor, left stick scrolls.
  • Auto-resume of a running app when Aurora returns to the foreground.
